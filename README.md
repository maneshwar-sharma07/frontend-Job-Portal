A fully responsive Job Portal Website built using HTML, CSS, and JavaScript. This platform allows users to explore job opportunities with advanced search and filtering, interact through an anonymous chat system, and stay updated with the latest industry news.
🚀 Features
🔍 Dynamic Job Search & Filtering
Real-time job search functionality
Filter jobs based on:
Location
Job type (Full-time, Part-time, Remote)
Category/Industry
Instant results without page reload (client-side filtering)
📄 Job Listings
Clean and structured job cards
Displays:
Job title
Company name
Location
Short description
Easy navigation and readability
💬 Anonymous Chat Section
Users can communicate without revealing identity
Useful for:
Asking queries about jobs
Peer discussions
Simple UI for sending and receiving messages
📰 News Section
Displays latest news related to:
Jobs
Technology
Career trends
Helps users stay updated with industry insights
📱 Responsive Design
Fully responsive across:
Mobile devices
Tablets
Desktops
Built using modern CSS techniques (Flexbox/Grid)
🛠️ Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Functionality and interactivity
